What is Sci-Flix?

Sci-Flix is a database and API of sci-fi movies, with subgenres!.

    Request: List of all movies

  • URL: http://localhost:8080/movies/
  • Method: GET
  • Response: JSON list of all movies
  • Example: http://localhost:8080/movies/
  • Example Response: 
    • {
    "title": "2001: A Space Odyssey",
    "year": "1968",
    "director": "Stanley Kubrick",
    "subgenre": "Epic",
    "description": "A visually stunning sci-fi epic exploring human evolution, AI, and cosmic mysteries in deep space."
},
{
    "title": "Alien",
    "year": "1979",
    "director": "Ridley Scott",
    "subgenre": "Horror",
    "description": "The crew of the Nostromo battles a terrifying alien life form in this suspenseful and claustrophobic horror classic."
}
...

    Request: Request movie Metadata

  • URL: http://localhost:8080/movies/:movieName/
  • Method: GET
  • Response: JSON object with movie title, year, director, subgenre, and description
  • Example Request: "http://localhost:8080/movies/Arrival/"
  • Example Response:
    • {
    "title": "Arrival", 
    "year": "2016", 
    "director": "Denis Villeneuve", 
    "subgenre": "Drama", 
    "description": "A linguist works to communicate with mysterious alien visitors and unravels profound implications for humanity and herself."
}

    Request: Get genre description by name

  • URL: http://localhost:8080/genres/:genreName/
  • Method: GET
  • Response: JSON object with genre description
  • Example Request: "http://localhost:8080/genres/Thriller/"
  • Example Response:
    • {
"name": "Thriller", 
"description": "A genre characterized by excitement, suspense, and high stakes, often involving crime or espionage."
}

    Request: Get director data by name

  • URL: http://localhost:8080/directors/:directorName/
  • Method: GET
  • Response: JSON object with director bio, birth year, and death year (if applicable)
  • Example Request: "http://localhost:8080/directors/ChristopherNolan/"
  • Example Response:
    • {
    "name": "Christopher Nolan", 
    "bio": "British-American film director, screenwriter, and producer, known for his cerebral, often nonlinear storytelling.",
    "birthYear": "1970",
    "deathYear": null
}

    Request: Register new user

  • URL: http://localhost:8080/users/
  • Method: POST
  • Body: JSON object with user information
  • Example Request Body:
    • {
    "username": "newUser",
    "password": "password123",
    "email": "newuser@example.com"
}
  • Example Response:
    • New user successfully registered

    Request: Update user info (username)

  • URL: http://localhost:8080/users/:username/
  • Method: PUT
  • Body: JSON object with updated user information
  • Example Request Body:
    • {
    "username": "updatedUsername"
}
  • Example Response:
    • User info updated successfully.

    Request: Add movie to user's list of favorites

  • URL: http://localhost:8080/users/:username/favorites/:movieName/
  • Method: POST
  • Response: JSON object with success message
  • Example Request: "http://localhost:8080/users/johndoe/favorites/Arrival/"
  • Example Response:
    • Movie successfully added to your favorites.

    Request: Remove movie from user's list of favorites

  • URL: http://localhost:8080/users/:username/favorites/:movieName/
  • Method: DELETE
  • Response: JSON object with success message
  • Example Request: "http://localhost:8080/users/johndoe/favorites/Arrival/"
  • Example Response:
    • Movie removed from your favorites.

    Request: Deregister user

  • URL: http://localhost:8080/users/:username/
  • Method: DELETE
  • Response: JSON object with success message
  • Example Request: "http://localhost:8080/users/johndoe/"
  • Example Response:
    • User successfully deleted
