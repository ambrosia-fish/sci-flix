Sci-Flix API Documentation

Welcome to the Sci-Flix API documentation. This guide provides information about available endpoints and examples for interacting with the Sci-Flix database of sci-fi movies.

API Base URL

All API requests should be made to: http://localhost:8080/ for local development or https://sci-flix.feztech.io/ for production.

Genre Endpoints

Request: Get genre description by name

  • URL: http://localhost:8080/movies/genres/:genreName/
  • Method: GET
  • Response: JSON object with genre description
  • Example Request: "http://localhost:8080/movies/genres/Epic/"
  • Example Response:
{
    "genreDescription": "Epic films are a style of filmmaking with large-scale, sweeping scope, and spectacle."
}

Director Endpoints

Request: Get director data by name

  • URL: http://localhost:8080/movies/directors/:directorName/
  • Method: GET
  • Response: JSON object with director bio, birth year, and death year (if applicable)
  • Example Request: "http://localhost:8080/directors/Ridley%20Scott/"
  • Example Response:
{
    "Name": "Ridley Scott",
    "Bio": "Sir Ridley Scott is an English film director and producer, known for his stylish visuals and innovative storytelling.",
    "Birth": "1937-01-01T00:00:00.000Z",
    "Death": null
}

User Endpoints

Request: Register new user

  • URL: http://localhost:8080/users/
  • Method: POST
  • Body: JSON object with user information
  • Example Request Body:
{
    "username":"NewUserTest123",
    "password":"supertoughpassword",
    "email":"first.last@gmail.com"
}
  • Example Response:
{
    "username": "NewUserTest123",
    "password": "supertoughpassword",
    "email": "first.last@gmail.com",
    "FavoriteMovies": [],
    "_id": "666369f556d09b268460cf97",
    "__v": 0
}

Request: Update user info (username)

  • URL: http://localhost:8080/users/:username/
  • Method: PUT
  • Body: JSON object with updated user information
  • Example Request Body:
{
    "username": "updatedUsername"
}
  • Example Response:
{
    "_id": "666369f556d09b268460cf97",
    "username": "UpdatedUserName",
    "password": "EvEnMoReStRoNgPaSsWoRd",
    "email": "first.last@gmail.com",
    "FavoriteMovies": [],
    "__v": 0
}

Request: Add movie to user's list of favorites

  • URL: http://localhost:8080/users/:username/favorites/:movieName/
  • Method: POST
  • Body: JSON object with newFavorite parameter with a movieId value
  • Example Request Body:
{
    "newFavorite":"6661eee0e9e6e3b6dacdcdf8"
}
  • Response: JSON object of user with updated favorited list
  • Example Request: "http://localhost:8080/users/ambrosia.fish/favorites/Alien/"
  • Example Response:
{
    "_id": "666359f3bebc0af4e6c1283d",
    "username": "ambrosia.fish",
    "password": "supertoughpassword",
    "email": "fish.ambrosia@gmail.com",
    "FavoriteMovies": [
        "6661eee0e9e6e3b6dacdcdff",
        "6661eee0e9e6e3b6dacdcdf8"
    ],
    "__v": 0
}

Request: Remove movie from user's list of favorites

  • URL: http://localhost:8080/users/:username/favorites/:movieName/
  • Method: DELETE
  • Response: JSON object of user with updated favorited list
  • Example Request: "http://localhost:8080/users/ambrosia.fish/favorites/Alien/"
  • Example Response:
{
    "_id": "666359f3bebc0af4e6c1283d",
    "username": "ambrosia.fish",
    "password": "supertoughpassword",
    "email": "fish.ambrosia@gmail.com",
    "FavoriteMovies": [
        "6661eee0e9e6e3b6dacdcdff",
    ],
    "__v": 0
}

Request: Deregister user

  • URL: http://localhost:8080/users/:username/
  • Method: DELETE
  • Response: JSON object with success message
  • Example Request: "http://localhost:8080/users/johndoe/"
  • Example Response:
User successfully deleted
